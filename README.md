# 🧪 A/B Testing Landing Page

## 📖 Description

This project analyzes an A/B testing experiment conducted on a landing page to determine whether a new design improves user conversion rates. Using Python, the project includes data cleaning, exploratory data analysis, hypothesis testing, and statistical validation to support data-driven decision-making.

---

## 🎯 Problem Statement

The company wanted to evaluate whether a redesigned landing page generated a higher conversion rate than the current version. Rather than relying on assumptions, statistical analysis was required to determine if the observed differences were significant enough to justify implementing the new design.

---

## 🛠️ Solution

To address this problem, I:

- Imported and validated the experiment dataset.
- Cleaned and prepared the data for analysis.
- Performed Exploratory Data Analysis (EDA).
- Compared conversion metrics between both user groups.
- Applied hypothesis testing to evaluate statistical significance.
- Interpreted the results and provided business-oriented conclusions.

---

## 💡 Recommendations

- Deploy the new landing page only if the improvement is statistically significant.
- Continue performing A/B tests before implementing major product changes.
- Analyze conversion rates by user segments to identify additional optimization opportunities.
- Monitor post-deployment performance to validate long-term results.
